# 🌡️ Temperature Conversion Program (Win32 GUI - C++)

## 📝 Task Overview

**Task-01: Build a Temperature Conversion Program**

Create a Windows GUI application using **C++** and the **Win32 API** that converts temperatures between Celsius, Fahrenheit, and Kelvin.

The program should:
- Prompt the user to enter a **temperature value**
- Let the user select the **original unit** (Celsius, Fahrenheit, or Kelvin)
- Convert the value to the **other two units**
- Display all results clearly in the GUI

---

## 🧠 Features

- 🎛️ Input field for temperature value
- 📌 Dropdown or buttons for selecting input unit
- 🔁 Automatically convert to other units
- 📋 Display all three values
- ✅ Input validation
- 🖥️ Fully implemented with **Win32 GUI**

## 🎯 Sample Conversion Logic

If input = **25°C**:

- Fahrenheit = `(25 × 9/5) + 32` = `77°F`
- Kelvin = `25 + 273.15` = `298.15 K`
